Git Versus Github

  • Git: version-control software
    • Lets you keep track of source code history
  • Github: web-based Git version control repository hosting service
    • Helps manage open-source projects that use Git

Now we'll set up our own Github project

Let's start by creating a github account.

Go to www.github.com and click "Sign Up" in the top right

Now we'll create a copy (or "fork") of the Github project

  • Go to https://github.com/Affirm/affirm-cares-github-workshop
  • Click the "Fork" button in the top right

If you see a popup asking you where to fork it, click your username.

After a minute or so, you should be taken to your new forked repo.
